# todo-list - v0.1.1

This package keeps track of TODO, FIXME, etc. messages in code.

![Main screenshot.](http://anthony-calandra.com/downloads/todo-list1.png)

## Features
 - Toggle Todo List
 - Resizable Panel
 - Double-click message block to jump to code
 - Left/Right Displayable Panel
